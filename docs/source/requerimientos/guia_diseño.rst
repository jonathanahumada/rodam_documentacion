####################
Guía de Diseño
####################
En estos documentos se recogen los artefactos de diseño para el proyecto.
Como se utiliza un marco de desarrollo ágil, los artefactos de diseño
se generan de acuerdo a su necesidad ('as needed').

Si se necesita más información acerca de qué es el diseño y como se
aplica al marco ágil, consultar: BRAUDE, Eric J.; BERNSTEIN,
Michael E. Software engineering: modern approaches. Waveland
Press, 2016.
