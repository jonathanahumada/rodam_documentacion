##############################
Flujo de emisión
##############################

Resumen
##############################

Este es un resumen de alto nivel de las vistas que existen y las
acciones que se pueden realizar para esta área de la aplicación.

¿Qué se hace en este flujo?
------------------------------

Las vistas de emisión sirven para *realizar acciones sobre la muestra
luego de que salen del proceso de recolección*. Por ejemplo:

1. aprobar
2. facturar
3. exportar

¿Cómo puedo comprender este flujo?
----------------------------------
En este segmento del proceso de la muestra, se pueden
realizar diferentes acciones en *orden más o menos
arbitrario*.

Para entender las acciones que se pueden realizar,
se puede utilizar el siguiente diagrama.

En este, se ponen las vistas (los rectángulos) y
se relacionan las acciones (los óvalos) que se pueden hacer
